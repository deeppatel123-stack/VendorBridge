# VendorBridge

Procurement & Vendor Management ERP.

## Project Structure

```
odoo-ksv/
├── frontend/     # React + Vite UI (all frontend code lives here)
└── README.md
```

## Quick Start (Frontend)

```bash
cd frontend
npm install
npm run dev
```

Then open **http://localhost:5173**

## Notes

- All frontend source code lives inside the `frontend/` folder.
